
                    CASE SEARCHER EXTENSION - QUICK START


A Chrome extension that automatically extracts and searches Customer IDs or 
Order Numbers on case management pages.


QUICK INSTALLATION


STEP 1: Double-click configure.bat

STEP 2: Enter your domain (example: cs.yourcompany.com)

STEP 3: Enter your case view path (example: inquiry-center/cases/view)

STEP 4: Open Chrome and go to: chrome://extensions/

STEP 5: Enable Developer mode (top right corner)

STEP 6: Click "Load unpacked" and select this folder


USAGE


1. Click the extension icon in Chrome toolbar
2. Select Customer or Order mode
3. Navigate to any case view page
4. Extension automatically extracts and searches
